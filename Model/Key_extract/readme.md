# This is keyword extraction project
### Usage
* python keyword_extraction.py \[file name\] \[save name\] -n \[sentence n\] -k \[top k\] -w \[with weight=1 or 0\]
	* \[file name\]: 原始路径
	* \[save name\]: 保存路径
	* \[sentence n\]: 每n句
	* \[top k\]: top k个关键词
	* \[with weight=1 or 0\]: 是否需要显示权重，实际设置/不设置对提取关键词无影响
